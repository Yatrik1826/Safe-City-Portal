# Nirbhaya Safe City Intelligent Monitoring & Response System

A research-grade MERN platform that simulates smart city safety operations for women’s safety. Includes incident reporting, realtime response simulation, analytics, and AI-lite prediction.

## Phased Execution Plan

1. Phase 1: Backend + Database + Simulation
2. Phase 2: APIs + Realtime system
3. Phase 3: Frontend UI
4. Phase 4: Analytics + Maps
5. Phase 5: Deployment

See `docs/ARCHITECTURE.md`, `docs/API.md`, and `docs/SETUP.md` for deep dives.
